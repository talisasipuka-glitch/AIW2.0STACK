# Agent: hero-image-agent (Stage 9)

## Role

Generate or finalize the hero image composition: real attorney photo (or
flagged placeholder), trust chips, and the 4-field LeadForm layout, in the
niche's restrained, professional mood. Runs BEFORE Stage 10.1.

## Prerequisites

Stage 7 (`brand-dna-agent.md`) and, if applicable, Stage 7.5
(`07-5-brand-resonance.md`) have completed.

## Inputs

- `clients/[Client Name]/Pipeline Data/brand/brand-dna.json`
- `clients/[Client Name]/[Client Name] Assets/`
- `clients/[Client Name]/Pipeline Data/brand-resonance/` (if Stage 7.5 ran)
- `nano-banana/SKILL.md`
- `taste/skills/imagegen-frontend-web/SKILL.md`
- `.claude/sops/08-hero-image.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/08-hero-image.md` and any per-client lessons
at `clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule
as an override to the spec below.

### Step 1, Check for a real attorney photo, compose around it

Follow `.claude/sops/08-hero-image.sop.md` Procedure steps 1 to 3: if a real
attorney headshot/team photo exists from Stage 4, compose the hero around
it. If not, generate a generic professional scene (not a synthetic invented
"attorney") and flag for the delivery report. Compose per `09-wireframe.md`:
split layout, copy+LeadForm left/center, attorney image right (image first
on mobile), trust chips above the H1.

### Step 2, Apply motion and palette restraint

Follow steps 4 to 7: apply restrained motion (600ms duration, 90ms stagger,
`prefers-reduced-motion` honored), palette restraint (yellow only on CTA,
"OPEN 24/7" badge, dollar figures), low-opacity (0.08) chevron corner
overlay if used, and verify above-the-fold composition at 1440x900 with the
LeadForm submit button visible.

## Pass criteria

Per `.claude/sops/08-hero-image.sop.md` Pass criteria: split layout
composition correct with proper mobile stacking; real attorney photo used
if available, else flagged generic scene; palette restraint holds; motion
preset restrained; above-the-fold check passes at 1440x900.

## When this agent halts

This agent does not hard-halt the pipeline, but per
`.claude/sops/08-hero-image.sop.md`, if no real attorney photo exists it
MUST write a flag to the delivery report (Stage 12) that carries through to
the proposal (Stage 13).

## Outputs

`clients/[Client Name]/Pipeline Data/hero-image/` populated with the
composed hero image and any follow-up flags. Hands off to `09-build.md`.
