# Agent: build-agent (Stage 10.1)

## Role

Clone the personal-injury-lawyers niche template, overlay the client's
brand-dna, build the site, and validate all 10 pages including all 7
case-type routes render correctly with the 4-field LeadForm.

## Prerequisites

Stage 7 (`brand-dna-agent.md`) and Stage 9 (`08-hero-image.md`) have
completed.

## Inputs

- `clients/[Client Name]/Pipeline Data/brand/brand-dna.json`
- `templates/personal-injury-lawyers/`
- `tools/build-from-template.py`
- `.claude/sops/09-build.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/09-build.md` and any per-client lessons at
`clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule as
an override to the spec below.

### Step 1, Clone template and fill brand-dna

Follow `.claude/sops/09-build.sop.md` Procedure steps 1 to 3: run
`tools/build-from-template.py` to clone
`templates/personal-injury-lawyers/` into
`clients/[Client Name]/[Client Name] Website/`, fill
`src/config/brand-dna.js` from `brand-dna.json`, and run the build.

### Step 2, Validate routes and components

Follow steps 4 to 8: validate the route list against `09-sitemap.json`
including every case-type route present in `services[]` (up to 7); confirm
each case-type page has non-empty H1, settlement figure or fallback, body,
and at least one FAQ item; confirm every LeadForm instance has exactly 4
fields; confirm footer renders license number, hours, and bar links;
confirm header CALL NOW link and OPEN 24/7 badge render and stay sticky.

## Pass criteria

Per `.claude/sops/09-build.sop.md` Pass criteria: build completes without
errors; all sitemap routes resolve including every covered case-type route;
no case-type page empty/missing H1/zero FAQ; every LeadForm has exactly 4
fields; footer renders license number and hours; header CALL NOW and OPEN
24/7 badge render and stay visible on scroll.

## When this agent halts

Per `.claude/sops/09-build.sop.md` When this stage halts: build fails or
template missing (halt with error pointing to `/build-niche-template`); a
covered case type has no route or an empty route; any LeadForm has a field
count other than 4; `company.licenseNumber` missing from footer.

## Outputs

`clients/[Client Name]/[Client Name] Website/dist/`. Hands off to
`10-personalize.md`.
