# Agent: build-fidelity-agent (Stage 10.4c)

## Role

Run a DOM structural diff between the client build and the niche template's
reference build, confirming the sitemap's 10 pages, including all
applicable case-type routes, match structurally.

## Prerequisites

Stage 10.4b (`sop-qa-agent.md`) has passed or been overridden.

## Inputs

- `clients/[Client Name]/[Client Name] Website/dist/`
- `templates/personal-injury-lawyers/` (reference build)
- `research/02-niche-research/personal-injury-lawyers/09-sitemap.json`
- Playwright (DOM walk)
- `.claude/sops/10-4c-build-fidelity.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/10-4c-build-fidelity.md` and any per-client
lessons at `clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply
every rule as an override to the spec below.

### Step 1, Diff DOM structure across all pages

Follow `.claude/sops/10-4c-build-fidelity.sop.md` Procedure steps 1 to 3:
build the reference template if needed; walk both DOM trees for all 10
sitemap pages and every covered case-type route; diff element hierarchy,
component boundaries, ARIA roles, and identifier attributes.

### Step 2, Check LeadForm, sticky CTA, and breadcrumb structure

Follow steps 4 to 7: confirm every LeadForm has exactly 4 input fields at
the DOM level; confirm the sticky CALL NOW element and OPEN 24/7 badge (if
applicable) are present with correct positioning; confirm breadcrumb
structure on every case-type page; report any structural divergence.

## Pass criteria

Per `.claude/sops/10-4c-build-fidelity.sop.md` Pass criteria: DOM structure
for every sitemap page matches the reference for that page type; every
covered case-type route matches the CaseTypeLandingPage reference structure
in order; every LeadForm has exactly 4 input fields; sticky header CTA
present; breadcrumb structure present on every case-type page.

## When this agent halts

Per `.claude/sops/10-4c-build-fidelity.sop.md` When this stage halts: a
covered case-type route is missing one or more of the 6
CaseTypeLandingPage sections or they are out of order; a LeadForm has a
DOM-level field count other than 4; homepage section order in the DOM does
not match `09-wireframe.md`.

## Outputs

Build fidelity report. On pass, hands off to `10-4d-perf.md`.
