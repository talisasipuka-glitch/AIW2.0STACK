# Agent: personalize-agent (Stage 10.2)

## Role

Inject SEO metadata and the full schema markup set (Attorney, LegalService,
LocalBusiness, AggregateRating/Review, FAQPage, BreadcrumbList) and generate
`sitemap.xml`.

## Prerequisites

Stage 10.1 (`09-build.md`) has completed.

## Inputs

- `clients/[Client Name]/[Client Name] Website/dist/`
- `clients/[Client Name]/Pipeline Data/seo/`
- `clients/[Client Name]/Pipeline Data/brand/brand-dna.json`
- `seo/SKILL.md`
- `.claude/sops/10-personalize.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/10-personalize.md` and any per-client lessons
at `clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule
as an override to the spec below.

### Step 1, Inject titles, meta, and core schema

Follow `.claude/sops/10-personalize.sop.md` Procedure steps 1 to 4: inject
per-page titles/meta using Stage 3's confirmed formulas; inject
`LegalService`/`Attorney` JSON-LD (including `company.licenseNumber`);
inject `LocalBusiness` JSON-LD with accurate hours (24/7 only if confirmed);
inject `AggregateRating`/`Review` schema.

### Step 2, Inject FAQ/Breadcrumb schema and sitemap

Follow steps 5 to 8: inject `FAQPage` schema on the homepage FAQ and every
case-type page's FAQ block; inject `BreadcrumbList` on every covered
case-type page; generate `sitemap.xml` covering all rendered routes;
validate all JSON-LD.

## Pass criteria

Per `.claude/sops/10-personalize.sop.md` Pass criteria: title/meta formulas
applied; `LegalService`/`Attorney`/`LocalBusiness` JSON-LD valid with
license number reflected; `AggregateRating`/`Review` schema present;
`FAQPage` on homepage and every case-type page; `BreadcrumbList` on every
case-type page; `sitemap.xml` complete; all JSON-LD valid.

## When this agent halts

Per `.claude/sops/10-personalize.sop.md` When this stage halts: malformed or
incomplete JSON-LD; `LocalBusiness` encodes "Open 24 hours" without Stage 1
confirmation; a case-type page missing `FAQPage` or `BreadcrumbList`.

## Outputs

Updated `clients/[Client Name]/[Client Name] Website/dist/` with injected
schema, meta, and `sitemap.xml`. Hands off to `11-uplift.md`.
