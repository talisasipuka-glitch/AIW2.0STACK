# Agent: proposal-agent (Stage 13)

## Role

Generate the final client-facing HTML proposal: delivered site summary,
competitive position grounded in Stage 2 data, and next steps, in the
serious, professional, trustworthy tone this niche's clients expect.

## Prerequisites

Stage 12 (`12-delivery.md`) has completed. Runs after deploy.

## Inputs

- `clients/[Client Name]/Pipeline Data/delivery/`
- `clients/[Client Name]/Pipeline Data/research/`
- `clients/[Client Name]/Pipeline Data/deploy/`
- `proposal-writing/SKILL.md`
- `impeccable/` skill
- `.claude/sops/14-proposal.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/14-proposal.md` and any per-client lessons at
`clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule as
an override to the spec below.

### Step 1, Summarize delivery and build the competitive argument

Follow `.claude/sops/14-proposal.sop.md` Procedure steps 1 to 3: summarize
delivered pages, production URL, and the case-type pages as the headline
SEO feature. Build the competitive argument from Stage 2's competitor data:
if local competitors lack case-type landing pages, state this plainly with
the `"[case type] lawyer [city]"` keyword pattern named. Summarize the
trust elements built (contingency placement, settlement StatsBar,
FounderStory, testimonials, full schema set).

### Step 2, List follow-ups and generate the proposal

Follow steps 4 to 6: list Stage 12's follow-up items as recommended next
steps, framed constructively. Generate the HTML proposal using
`proposal-writing/SKILL.md`'s voice and structure. Do not include
unapproved pricing or contract terms.

## Pass criteria

Per `.claude/sops/14-proposal.sop.md` Pass criteria: proposal summarizes
delivered pages, URL, and case-type advantage with the specific keyword
pattern cited; competitive argument grounded in real Stage 2 data; trust
elements summarized accurately; follow-ups framed as next steps; no
unapproved pricing/terms; copy-lint passes.

## When this agent halts

Per `.claude/sops/14-proposal.sop.md` When this stage halts: Stage 11 has
not completed (no production URL); Stage 2 found no usable competitor data
(omit the competitive comparison rather than inventing one).

## Outputs

`clients/[Client Name]/[Client Name] Proposal/` with the final HTML
proposal. Pipeline complete for this client.
