# Agent: design-fidelity-qa-agent (Stage 10.4a)

## Role

Score the built site's visual fidelity against the niche template's
reference render via SSIM, region by region, per the per-niche checklist
and the weights in `09-wireframe.md`.

## Prerequisites

Stage 10.3 (`11-uplift.md`) has completed.

## Inputs

- `clients/[Client Name]/[Client Name] Website/dist/`
- `templates/personal-injury-lawyers/.claude/checklists/design-fidelity.md`
- Playwright (screenshots)
- `impeccable/` skill
- `.claude/sops/design-fidelity-qa-agent.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/design-fidelity-qa-agent.md` and any
per-client lessons at
`clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule as
an override to the spec below.

### Step 1, Capture and compare

Follow `.claude/sops/design-fidelity-qa-agent.sop.md` Procedure steps 1 to
2: capture desktop and mobile screenshots of Home, each rendered case-type
page, and the global Header/Footer, and the corresponding reference
renders.

### Step 2, Score regions against thresholds

Follow steps 3 to 6: run SSIM per region against
`.claude/checklists/design-fidelity.md`'s thresholds (Hero 0.95, StatsBar
0.92, Header 0.92, PracticeAreasGrid 0.90, ProcessSteps/CTABand/Footer
0.88, PressBand/WhyChooseUs/FounderStory/Testimonials/FAQSection 0.85;
case-type page CaseTypeHero 0.92, StatsBar single-figure 0.90, RiskRemoval/
CaseTypeExplainer/FAQSection/CTABand+LeadForm 0.85-0.88). Compute weighted
means against the 0.90 target band (~0.895 Home, ~0.884 Case-Type). For any
region below threshold, identify the specific visual delta and route back
to Stage 10.1/10.3 with a specific finding.

## Pass criteria

Per `.claude/sops/design-fidelity-qa-agent.sop.md` Pass criteria: every
region meets its threshold, or the weighted mean meets its target band with
no region more than 0.03 below threshold; global Header/Footer meet
thresholds across the whole site.

## When this agent halts

Per `.claude/sops/design-fidelity-qa-agent.sop.md` When this stage halts: a
region more than 0.03 below threshold with no clear remediation (escalate
via `/override-design-fidelity`); Hero region failure should not be
overridden without explicit student review.

## Outputs

QA report with per-region scores and any routed-back findings. On pass,
hands off to `sop-qa-agent.md`.
