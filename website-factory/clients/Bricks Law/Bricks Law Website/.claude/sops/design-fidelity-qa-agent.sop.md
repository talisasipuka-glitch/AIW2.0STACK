# 10.4a - Design Fidelity QA

Implements: Stage 10.4a.

## Purpose

Score the built site's visual fidelity against the niche template's
reference render via SSIM, region by region, per the weights in
`09-wireframe.md`.

## Inputs (per-client)

```
clients/[Client Name]/[Client Name] Website/dist/
templates/personal-injury-lawyers/.claude/checklists/design-fidelity.md
```

## Procedure

1. Capture screenshots of the built site (desktop + mobile) for: Home,
   each rendered case-type page, and the global Header/Footer.
2. Capture the corresponding reference renders from the niche template.
3. Run SSIM comparison region by region per
   `.claude/checklists/design-fidelity.md` (generated from
   `09-wireframe.md`'s region table), applying the documented thresholds:
   Header 0.92, Hero 0.95, ProcessSteps 0.88, StatsBar 0.92, PressBand 0.85,
   WhyChooseUs 0.85, PracticeAreasGrid 0.90, FounderStory 0.85,
   Testimonials 0.85, FAQSection 0.85, CTABand 0.88, Footer 0.88.
4. For the case-type page template, apply: CaseTypeHero 0.92,
   RiskRemovalBand 0.85, StatsBar (single-figure) 0.90, CaseTypeExplainer
   0.85, FAQSection 0.85, CTABand+LeadForm 0.88.
5. Compute the weighted mean per page type and compare against the 0.90
   target band (Home target ~0.895, Case-Type target ~0.884, per
   `09-wireframe.md`).
6. For any region scoring below threshold, identify the specific visual
   delta (layout shift, color drift, spacing, missing element) and route
   back to Stage 10.1/10.3 with a specific finding.

## Pass criteria

- Every region meets or exceeds its documented threshold, OR the weighted
  mean for the page type meets its target band (~0.895 Home, ~0.884
  Case-Type) with no single region more than 0.03 below its threshold.
- Header and Footer (global regions) meet their thresholds across the
  whole site, not per-page.

## When this stage halts

- A region scores more than 0.03 below threshold with no clear remediation
  (e.g. a structural mismatch the build cannot resolve). Escalate to the
  student via `/override-design-fidelity`.
- The Hero region (weight 0.20, threshold 0.95) fails: this is the
  highest-weighted region in the template and should not be overridden
  without explicit student review, given its role as first impression for
  a stressed, comparison-shopping visitor.

## Niche-specific notes

`research/02-niche-research/personal-injury-lawyers/09-template-spec.md` +
the niche playbook):
