# Agent: brand-resonance-agent (Stage 7.5)

## Role

Analyze the client's existing website (if any) for elements to preserve or
intentionally drop, balanced against this niche's required restrained,
serious, professional direction.

## Prerequisites

Stage 7 (`brand-dna-agent.md`) has completed. This stage is optional and
runs only when an existing client website URL is present from Stage 1
intake.

## Inputs

- `clients/[Client Name]/Pipeline Data/brand/brand-dna.json`
- Client's existing website URL (from Stage 1 intake, if present)
- `.claude/sops/07-5-brand-resonance.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/07-5-brand-resonance.md` and any per-client
lessons at `clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply
every rule as an override to the spec below.

### Step 1, Check for existing site, skip if absent

Follow `.claude/sops/07-5-brand-resonance.sop.md` Procedure step 1: if no
existing website URL was captured at Stage 1, write "no existing site,
skipped" to `Pipeline Data/brand-resonance/` and stop.

### Step 2, Capture and compare

Follow steps 2 to 6: capture desktop and mobile screenshots, extract
palette/typography/distinctive elements, and compare against this niche's
required direction (light theme, near-black + yellow-accent-only palette,
chevron motif, restrained 600ms motion). Flag elements to preserve (real
brand continuity, e.g. recognizable logo color or tagline) and elements to
intentionally drop (gavel/scales-of-justice imagery, dark aggressive theme,
overcrowded multi-CTA layout).

## Pass criteria

Per `.claude/sops/07-5-brand-resonance.sop.md` Pass criteria: either a
documented "skipped" record, or a findings document covering palette/
typography comparison, elements to preserve, and elements intentionally
dropped.

## When this agent halts

This agent does not halt the pipeline. It is informational input to Stage 9
and Stage 11.

## Outputs

`clients/[Client Name]/Pipeline Data/brand-resonance/` populated (or
"skipped" record). Hands off to `08-hero-image.md`.
