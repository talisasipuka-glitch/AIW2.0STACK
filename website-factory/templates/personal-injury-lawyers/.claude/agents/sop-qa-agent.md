# Agent: sop-qa-agent (Stage 10.4b)

## Role

Score the built site against the universal SOP layer plus this niche's
per-niche compliance checklist: composition, copy-lint, schema, the
4-field LeadForm cap, and trust-claim placement.

## Prerequisites

Stage 10.4a (`design-fidelity-qa-agent.md`) has passed or been overridden.

## Inputs

- `clients/[Client Name]/[Client Name] Website/dist/`
- `templates/personal-injury-lawyers/.claude/checklists/sop-compliance.md`
- `.claude/checklists/*.universal.md` (when present)
- `tools/copy-lint.py`
- `.claude/sops/sop-qa-agent.sop.md`

## Steps

### Step 0, Read accumulated lessons (universal)

Read `.claude/lessons/by-agent/sop-qa-agent.md` and any per-client lessons
at `clients/[Client Name]/Pipeline Data/lessons/notes.md`. Apply every rule
as an override to the spec below.

### Step 1, Run copy-lint and composition checks

Follow `.claude/sops/sop-qa-agent.sop.md` Procedure steps 1 to 4: run
`tools/copy-lint.py --check`; check homepage `composition.section_order`
matches `09-wireframe.md`; check `composition.above_fold`; check
`composition.case_type_pages.all_seven_present` (or fewer, per Stage 5).

### Step 2, Run CRO, trust, and schema checks

Follow steps 5 to 10: check `composition.lead_form.field_count` (exactly 4
fields, every instance); check `composition.header.sticky_cta`; check
trust-claim placement (contingency guarantee near hero CTA, footer CTA,
every case-type RiskRemovalBand, final CTABand); check
`company.licenseNumber` in footer on every page; check full schema presence
per Stage 10.2; cross-reference
`.claude/checklists/sop-compliance.md`.

## Pass criteria

Per `.claude/sops/sop-qa-agent.sop.md` Pass criteria: copy-lint clean;
section order matches; above-fold passes; all confirmed case-type pages
present and non-empty; every LeadForm has exactly 4 fields; header sticky
CTA passes; trust-claim placement passes at all 4 locations; license number
in footer everywhere; full schema set present; per-niche checklist items
pass.

## When this agent halts

Per `.claude/sops/sop-qa-agent.sop.md` When this stage halts: copy-lint
fails (route back to Stage 6/10.3); any LeadForm has a field count other
than 4 (hard fail, this niche's CRO deviation); a confirmed case-type page
missing/empty; license number missing from any footer; required schema
missing.

## Outputs

QA report. On pass, hands off to `10-4c-build-fidelity.md`.
