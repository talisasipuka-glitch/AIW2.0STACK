# 7.5 - Brand Resonance

Implements: Stage 7.5 (optional, runs when an existing client website URL is
present).

## Purpose

Analyze the client's existing website (if any) for visual and tonal cues
that should carry forward into the new build, balanced against this
niche's required serious/professional/restrained direction.

## Inputs (per-client)

```
clients/[Client Name]/Pipeline Data/brand/brand-dna.json
Client's existing website URL (from Stage 1 intake, if present)
```

## Procedure

1. If no existing website URL was captured at Stage 1, skip this stage and
   record "no existing site, skipped" in `Pipeline Data/brand-resonance/`.
2. If a URL exists, capture screenshots (desktop + mobile) via Apify or
   Playwright.
3. Extract the existing site's color palette, typography, and any
   distinctive visual elements (logo treatment, photography style, motion).
4. Compare against this niche's required direction: light theme,
   near-black primary, bright-yellow accent reserved for CTAs and dollar
   figures only, chevron shape motif, restrained motion
   (`--motion-duration: 600ms`), per `09-template-spec.md` Sections 1 to 3.
5. Flag any strong existing brand element (a recognizable logo color, a
   tagline, a specific photography style already in use) that should be
   preserved even where it diverges from the template defaults. Brand
   continuity for an existing firm with name recognition can outweigh a
   template default.
6. Flag any existing element that actively works against the niche
   direction (e.g. gavel/scales-of-justice imagery, a dark "aggressive"
   theme, an overcrowded multi-CTA layout). These should NOT be carried
   forward; record them as "intentionally not preserved" with a one-line
   reason.
7. Write findings to `Pipeline Data/brand-resonance/`.

## Pass criteria

- Either a documented "skipped, no existing site" record, or a findings
  document covering: palette/typography comparison, elements to preserve,
  elements to intentionally drop.

## When this stage halts

- This stage does not halt the pipeline. It is informational input to
  Stage 9 (hero image) and Stage 11 (uplift). If the existing site's brand
  is in active legal/trademark dispute or otherwise unusable, note it and
  proceed with template defaults.

## Niche-specific notes

For this niche, the following pattern applies (derived from
`research/02-niche-research/personal-injury-lawyers/09-template-spec.md` +
the niche playbook):

Brand continuity matters more for an established firm with local name
recognition (a long-running local firm's existing color or logo may be a
real asset). However, this niche's template direction is deliberately
restrained (light theme, near-black + yellow-accent-only palette, chevron
motif, restrained motion) as a corrective to the niche's common
overcrowded, gavel-heavy, multi-CTA-repetition pattern
(`08-starter-template.md`). When an existing site's brand conflicts with
this direction, prefer preserving the LOGO and any genuinely distinctive
tagline or founder framing, while letting the template's palette, motion,
and shape motif govern the new build.
