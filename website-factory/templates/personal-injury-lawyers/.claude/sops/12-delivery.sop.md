# 12 - Delivery

Implements: Stage 12.

## Purpose

Produce the delivery report: what was built, what passed QA, what was
flagged for follow-up (e.g. a placeholder attorney photo from Stage 9), and
the production URL.

## Inputs (per-client)

```
clients/[Client Name]/Pipeline Data/deploy/
clients/[Client Name]/Pipeline Data/ (all prior stage outputs)
```

## Procedure

1. Compile a summary of the build: pages delivered (homepage + confirmed
   case-type pages + interior pages), production URL, and the QA results
   from Stages 10.4a to 10.4d.
2. Compile a follow-up list from flags raised during the pipeline:
   - Stage 4/9: if a real attorney headshot was unavailable and a
     placeholder/generic scene was used in the hero, list this as the
     top-priority follow-up item.
   - Stage 5: any case type the client does not currently cover but could
     add a page for later.
   - Stage 6: any settlement figure gaps (case types with no disclosable
     figure).
   - Any QA override granted via `/override-design-fidelity` or
     `/override-sop`.
3. Write the delivery report in the serious, professional tone this niche's
   client-facing communications use, per `09-template-spec.md` Section 1.
4. Confirm `company.licenseNumber`, contact details, and hours are correct
   in the delivered site as a final sanity check (these are
   client-liability-sensitive fields).

## Pass criteria

- Delivery report lists all delivered pages and the production URL.
- Follow-up list is complete and prioritized, with the attorney-photo gap
  (if any) flagged first.
- License number, contact details, and hours are confirmed correct.

## When this stage halts

- `company.licenseNumber`, phone, address, or hours in the delivered site
  do not match Stage 1 intake. This is a liability-relevant discrepancy;
  halt and correct before the report goes to the client.

## Niche-specific notes

For this niche, the following pattern applies (derived from
`research/02-niche-research/personal-injury-lawyers/09-template-spec.md` +
the niche playbook):

The delivery report's follow-up list should treat a missing real attorney
photo as the single highest-priority item, given how much trust weight this
niche places on a named, photographed attorney (`05-trust-signals.md`).
Beyond that, the report follows the standard factory delivery format with
no other niche-specific structure changes. Tone matches the serious,
professional voice used throughout the site's copy: direct, factual,
no overstatement of what was delivered.
