# Factory blueprint

Universal scaffold the niche factory generator (Module 2D's
`/build-niche-template` orchestrator) stamps into every per-niche
template. Nothing here encodes a wireframe, a section composition, or a
visual decision. All files in this tree are niche-agnostic build
infrastructure.

## What lives here

```
factory-blueprint/
  scripts/
    inject-theme.mjs              prebuild hook: rewrites :root palette,
                                  Google Fonts imports, <html data-theme-
                                  mode>, <title>, <meta description>, and
                                  JSON-LD script from brand-dna.js.
                                  Niche-agnostic.
    validate-brand-dna.mjs        prebuild hook: walks the canonical
                                  shape contract at
                                  src/config/brand-dna.example.js and
                                  halts on missing fields, wrong types,
                                  or surviving sentinels (strict mode).
                                  Niche-agnostic.

  src/
    main.jsx                      React entry point. Universal.
    index.css.template            Foundation utilities + per-niche
                                  palette slots filled at generation.
    App.jsx.template              Router shell. Module 2D fills the
                                  routes from the niche sitemap.
    config/
      brand-dna.example.js        STAMPED FROM
                                  `references/brand-dna.shape.js`. The
                                  niche template's copy of the canonical
                                  shape contract.

  public/
    badges/                       Stage 4 fills this per client.
    work/                         Stage 4 fills this per client.

  checklists/
    sop-compliance.skeleton.md    Skeleton + {{wireframe_section}} slots.
                                  Module 2D fills per-section checks
                                  from the niche wireframe.
    design-fidelity.skeleton.md   Skeleton + {{region}} + {{weight}} +
                                  {{threshold}} slots. Module 2D fills
                                  region weights from the niche
                                  wireframe.

  sops/
    _stage-skeleton.md            Universal Stage SOP frame with
                                  {{niche_specifics}} slots Module 2D
                                  fills for each pipeline stage.

  agents/
    _stage-agent-skeleton.md      Universal Stage Agent frame. Module
                                  2D fills the per-stage Claude
                                  instructions per niche.
```

## How the generator uses this

When Module 2D's `/build-niche-template` runs:

1. Resolves the active niche from `stack-state.json`.
2. Copies the scripts + src/main.jsx + src/index.css.template +
   src/App.jsx.template into `templates/{niche-slug}/`, with token
   substitutions.
3. Stamps `references/brand-dna.shape.js` into
   `templates/{niche-slug}/src/config/brand-dna.example.js` plus a
   sentinel-filled `brand-dna.js`.
4. Generates the niche's tailwind.config.js + index.html from
   niche-design-tokens.json (via
   `tools/generate-build-skeleton.py`).
5. Generates per-section React components + pages by orchestrating
   `frontend-design` + `impeccable` + `ui-ux-pro-max` skills against
   the niche wireframe (Claude does this at runtime).
6. Generates the niche playbook by filling the schemas at
   `references/niche-playbook/schemas/` from niche research.
7. Generates the per-niche SOPs by filling each `sops/_stage-skeleton.md`
   slot from the niche wireframe + niche playbook.
8. Generates the per-niche QA checklists by filling
   `checklists/sop-compliance.skeleton.md` +
   `checklists/design-fidelity.skeleton.md` with the niche's actual
   components + regions.
9. Writes the niche template manifest to
   `templates/{niche-slug}/MANIFEST.json`.
10. Runs the 6-gate validator. Halts on any hard-gate failure
    (writes `templates/{niche-slug}/GENERATION-FAILED.md`).

## What stays universal

The build scripts (`inject-theme.mjs`, `validate-brand-dna.mjs`) are
genuinely niche-agnostic. They read brand-dna.js and mutate CSS/HTML.
They ship verbatim to every per-niche template.

The canonical brand-dna shape contract (`references/brand-dna.shape.js`)
defines the data API every niche template's components read from. The
shape is universal; only per-niche values vary.

The pipeline tools (`tools/build-from-template.py`, `tools/build-proposal.py`,
etc.) are universal orchestration. They consume per-niche templates,
but they themselves don't encode any wireframe.

## What's per-niche-generated

Everything in the niche template's `src/components/`, `src/pages/`,
`niche-playbook/`, `.claude/checklists/` is generated by Module 2D
per niche from the captured research + the four design skills. No
shared baseline; if generation fails, the factory halts.
