# Research Extensions, Personal Injury Lawyers

Written to unblock Stage 2 (Research) of `/run-factory`. Synthesized from
`research/02-niche-research/personal-injury-lawyers/` (sub-tasks 01-07 plus
synthesis.md) and the niche playbook already generated by Module 2D
(`trust-signals.json`, `vocabulary.json`, `copy-locks.json`).

## Directories to search (Step 6, third-party lookups)

- State Bar Association attorney directory (state-specific, e.g. "State Bar
  of Georgia attorney directory")
- AVVO (avvo.com)
- Martindale-Hubbell
- Super Lawyers (superlawyers.com)
- Justia
- FindLaw
- BBB
- Million Dollar Advocates Forum member directory
- National Trial Lawyers Top 100 directory

Populate `thirdPartyReviews` with one entry per directory found (rating,
review count, accreditation/membership status, profile URL).

## Niche-specific signals to detect (Step 11)

Populate `nicheExtensions["personal-injury-lawyers"]` per
`research-extensions.schema.json`.

### Practice profile

- `caseTypesHandled[]`: from {car-accident, truck-accident,
  motorcycle-accident, slip-and-fall, wrongful-death, dog-bite,
  brain-injury, medical-malpractice, workers-comp, product-liability,
  premises-liability, other}. List every case type the firm's site or
  listings mention.
- `contingencyFee`: `{ offered: boolean, termsText: string | "Not found" }`.
  Look for "No Win, No Fee", "no fee unless we recover", "free unless we
  win".
- `freeConsultation`: boolean. Almost universal in this niche, but confirm.
- `availabilityClaim`: `"24/7" | "business hours" | "Not found"`.
- `bilingualServices[]`: languages offered besides English (e.g. "Spanish").
  Check for "Se Habla Espanol" or `/es/` site paths.
- `officeLocations`: `{ count: number, cities: string[] }`. Single-office
  vs multi-office matters for the location-pages decision downstream.
- `homeHospitalVisits`: boolean. Some firms offer to visit injured clients
  who can't travel.

### Credentials & awards

- `barAdmissions[]`: states where the attorney(s) are licensed to practice.
- `yearsLicensed`: number, or `"Not found"`.
- `professionalAffiliations[]`: array of
  `{ name: string, tier: string | null, yearOrRating: string | null }`
  drawn from {Million Dollar Advocates Forum, National Trial Lawyers Top
  100, Super Lawyers, Best Lawyers, AVVO Rating, ABOTA, State Bar
  Association member, BBB Accredited}.
- `pressMentions[]`: "As Seen In" media logos or named outlets (local news,
  etc.).

### Results / proof

- `settlementResults[]`: array of
  `{ caseType: string, amountDisplay: string, year: string | null,
  description: string }`. Never fabricate, only record figures actually
  published by the firm or a directory profile.
- `testimonialThemes[]`: top 3-5 recurring phrases from reviews. This niche
  consistently surfaces: "kept me updated", "treated me like family",
  "explained every step", "fought for the best outcome".

### Business model classification

- `businessModel.primaryFocus`: `"contingency" | "hourly" | "hybrid"`.
  Default to `"contingency"` for PI firms unless the site states otherwise.
- `marketSegment.primaryFocus`: `"individual-injury" | "mass-tort" | "both"`.

### Special offers

- `specialOffers.freeConsultation`: boolean
- `specialOffers.noFeeUnlessWeWin`: boolean
- `specialOffers.bilingual`: boolean
- `specialOffers.virtualConsultation`: boolean
- `specialOffers.homeHospitalVisits`: boolean

All five must be explicitly `true` or `false`, never null, per the Stage 2
pass gate.

### Financing

- `financing.offered`: boolean. Rare in this niche since contingency fee
  itself is the financing mechanism. Default `false` unless the site
  explicitly mentions case-expense advances or litigation funding partners.

## Brand archetype tags for this niche

Tag `brandVoiceTags` with 3-5 from: `fighter`, `protector`, `compassionate`,
`accessible`, `aggressive`, `trustworthy`, `local`, `family-oriented`,
`results-driven`. Cross-reference against
`niche-playbook/vocabulary.json` and `copywriting.md` Section 1 (voice
grammar) for this niche's locked voice register ("commercial": serious,
direct, personally accountable).
